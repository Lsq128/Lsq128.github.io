window中常用的Doc命令

2020-04-04

自己整理的一些常用的命令

  • dir 显示一个目录下的文件和子目录
  • cd.. 返回上级目录v
  • cd/ 返回根目录
  • cd 进入一个目录
  • md/mkdir 创建一个子目录
  • rd/rmdir 删除目录
  • type 查看文件内容
  • cls 清屏
  • copy 复制一个文件到某个目录下
  • move 移动某个文件到某个目录下
  • echo 显示消息
  • ver 显示windows版本
  • exit 退出cmd.exe程序（命令翻译程序）或者当前批处理脚本
